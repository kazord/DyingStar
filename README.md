# StarDeception

### Controls
- [Z][S][Q][D] = move
- [Shift] = hold to sprint
- [C] = toggle crouch or slide (when sprinting)
- [Space] = jump, hold near ledge while falling to grab


### Code Organization

- addons: Godot Addons
- assets: Materials, Models, Music, Textures
- levels: Game Levels
- scenes: Other scenes
- server: Server only script/scenes
- test: Unit tests
- ui: UI Scenes
